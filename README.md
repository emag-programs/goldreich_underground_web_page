# Goldreich Underground - emag-programs Repository

This repository contains projects for electromagnetic analysis and engineering programs.

## Projects

### 1. Goldreich Underground Web Page
The main company website with information about services and contact details.

**Files:**
- `index.html` - Home page
- `about.html` - About page
- `Contact.html` - Contact information
- `goldreich_underground_logo_final.png` - Company logo
- `goldreich_photo.jpg` - Company photo

**Website URL:** https://emag-programs.github.io/goldreich_underground_web_page/

### 2. T3FEM v0 Project
A computational electromagnetics project for finite element analysis using triangular 3-node elements.

**Location:** `t3fem_v0/`

**Documentation:** See [t3fem_v0/README.md](t3fem_v0/README.md) for detailed information.

**Quick Start:** See [t3fem_v0/QUICK_START.md](t3fem_v0/QUICK_START.md) for instructions on adding your Visual Studio project files.
