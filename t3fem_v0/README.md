# T3FEM v0

## Overview
T3FEM (Triangular 3-Node Finite Element Method) version 0 is a computational electromagnetics project for finite element analysis.

## Project Structure

```
t3fem_v0/
├── src/           # Source code files
├── include/       # Header files
├── docs/          # Documentation
├── tests/         # Unit and integration tests
├── examples/      # Example programs and use cases
└── build/         # Build output directory (not tracked in git)
```

## Getting Started

### Prerequisites
- Visual Studio 2019 or later
- C++ compiler with C++17 support
- (Add other dependencies as needed)

### Building the Project

1. Open the Visual Studio solution file
2. Build the solution (Ctrl+Shift+B)
3. Run tests to verify the build

### Adding Files from Visual Studio

To add your Visual Studio project files to this structure:

1. Copy your `.cpp` files to the `src/` directory
2. Copy your `.h` or `.hpp` files to the `include/` directory
3. Copy your Visual Studio solution (`.sln`) and project (`.vcxproj`) files to this root directory
4. Copy any documentation to the `docs/` directory
5. Copy test files to the `tests/` directory
6. Copy example programs to the `examples/` directory
